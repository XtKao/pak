# Phalangnaukrian Party Campaign Website

This is a single-page website created for a high school student council election campaign (Candidate Number 2). 

## Educational Purpose
This project is built for educational purposes to demonstrate how to create a responsive and interactive static website using basic web technologies. It shows how to build a clean UI without relying on heavy frameworks, backend databases, or complex build tools.

## Tech Stack
- **HTML5:** Semantic structure.
- **Tailwind CSS (CDN):** Utility-first styling for responsive layouts.
- **Vanilla JavaScript:** DOM manipulation for a mock form submission and toast notifications.
- **AOS (Animate On Scroll):** Lightweight library for scroll reveal animations.

## Key Features
- Fully responsive layout (Mobile & Desktop).
- Smooth scroll animations.
- Interactive UI components (e.g., a fake submit contact form with a success notification popup).

## How to Run
Simply open the `index.html` file in any modern web browser. No installation or setup is required. Note that an internet connection is needed to load the CDN links (Tailwind CSS, AOS, and icons).

## Disclaimer
The contact form is for frontend demonstration purposes only. It uses a "fake submit" JavaScript function and does not send data to any server.
